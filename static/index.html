iOS Shortcut Setup Guide

Follow these steps to set up automatic saving of images to your Camera Roll:

  1. Download the "Shortcuts" app from the App Store if you haven't already
  2. Open the Shortcuts app and tap the + button to create a new shortcut
  3. Add "Select Files" as the first action
  4. Add "Extract Archive" as the second action
  5. Add "Filter Files" as the third action
  6. In the Filter Files action, set condition to "File Extension is png"
  7. Add "Save to Photo Album" as the final action
  8. Tap the ‚ãØ button at the top and name your shortcut (e.g., "Save Slides")
  9. Then click on the "i" icon at the top and pick the "add to homescreen" option

To use the shortcut:

  1. Click "Download All" in SlideSnap to get the ZIP file
  2. Once that is downloaded, open the new Shortcut on your homescreen and select your newly created zip file
  3. The images will be automatically saved to your Photos
Note: This shortcut only needs to be set up once and will work for all future downloads. The ZIP file will have the same name as your original document for easy identification.
